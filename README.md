# Furniture Sales Overview Dashboard

## Project Overview
This project focuses on a comprehensive analysis of furniture sales data across the U.S. using **Advanced Microsoft Excel**. I transformed raw sales data into a dynamic Business Intelligence (BI) dashboard to track key performance indicators (KPIs) like total sales, profit, and quantity sold.

## Dashboard Features
* **Sales & Profit Tracking:** Real-time summary of financial health and growth percentages.
* **Regional Analysis:** Map-based visualization of shipping distribution by state.
* **Logistics Insights:** Analysis of shipping modes (Standard Class being the most used at 58%) and shipping durations.
* **Product Performance:** Breakdown of highest-selling categories, such as Chairs ($101.8K).
* **Interactivity:** Integrated slicers for filtering by Segment (Consumer, Corporate, Home Office) and Region (Central, East, South, West).
